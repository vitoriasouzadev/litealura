📚 Literalura - Catálogo de Livros
Projeto desenvolvido como parte do desafio da Alura com o objetivo de criar um catálogo de livros com interação via console.

🎯 Objetivo
Desenvolva uma aplicação Java com Spring Boot que:

Busque livros através da API Gutendex
Salve os dados no banco de dados
Permitir interação textual via console
Oferecer no mínimo 5 opções de interação
🚀 Funcionalidades
Buscar livro pelo título
Listar livros registrados
Listar autores registrados
Listar autores vivos em determinado ano
Listar livros por idioma
🛠️ Tecnologias utilizadas
Java 17+
Bota de mola
Spring Data JPA
Hibernar
Banco H2
API Gutendex
🌐 API Utilizada
API Gutendex
https://gutendex.com/

🗄️ Banco de Dados
Banco em H2
Relacionamento ManyToOne entre Livro e Autor
▶️Como executar o projeto
Clonar onta
Abrir na IDE (IntelliJ)
Executar uma classeLitealuraApplication
Interagir com o menu via console
